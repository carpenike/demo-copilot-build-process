# Governance Exceptions

This directory contains approved exceptions to the enterprise engineering
standards defined in `governance/enterprise-standards.md`.

## Exception Process

1. An ADR must be written documenting the business justification
2. VP Engineering approval is required
3. The approved exception is stored here as a markdown file
4. Agents check this directory before flagging a governance conflict

## File Naming Convention

```
YYYY-MM-DD-<project>-<exception-summary>.md
```

Example: `2026-03-15-data-platform-rust-ml-inference.md`
